# 용어 저의
>Epic : 프로그램의 큰 틀
>Story : 발생하는 상황에 대한 스토리 (스프린트 단위로 구성하는게 좋다)
>Task : Story의 기능을 구현하기위해 필요한 작업
>sub-task : Task의 하위 작업내용들
>
>사용자 : 프로그램을 실행하는 우리
>client : Server에 접속한 client
>server : Webserver
