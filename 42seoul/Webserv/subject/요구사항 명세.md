# 용어 저의
>Epic : 프로그램의 큰 틀
>User Story : 발생하는 상황에 대한 스토리 (스프린트 단위로 구성하는게 좋다)
>Task : User Story의 기술적, 관리적 업무
>Sub-Task : User Story, Task를 더 작은 단위로 나눈 업무
>
>사용자 : 프로그램을 실행하는 우리
>client : Server에 접속한 client
>server : Webserver
