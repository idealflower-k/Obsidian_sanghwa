# Software Requirement Specification
- 소프트웨어의 요구사항을 분석하고 정의하는 단계에서 작성되는 최종 산출물이다.
- 소프트웨어가 수행해야 할 모든 기능과 제약 조건 증이 기술되어 있다.
- 소프트웨어 개발과 테스팅의 완료 승인에 대한 기준으로 사용된다.
- IEEE Std. 830-1998
