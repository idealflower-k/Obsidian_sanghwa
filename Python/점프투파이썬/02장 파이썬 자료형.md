## 8진수와 16진수
- 8진수(octal)를 만들기 위해서는 숫자가 0o 또는 0O(숫자 0 + 알파벳 소문자 o 또는 대문자 O)으로 시작하면 된다.
``` python
a = 0o177
print(a)
```
- 16진수(hexadecimal)를 만들기 위해서는 0x로 시작하면 된다.
``` python
>>> a = 0x8ff
>>> b = 0xABC
>>> print(b)
2748
```
## 제곱을 나타내는 ** 연산자
- (x ** y) x의 y제곱을 리턴한다.
``` python
>>> a = 3
>>> b = 4
>>> a ** b
81
```
## 나눗셈 후 몫을 리턴하는 // 연산자
``` python
>>> 7 // 4
1
```
